# 批量图片压缩 (batch-image-compress)

本地批量图片压缩工具，基于 Mulby 内置 Sharp 引擎，无需联网。

## 功能特性

- 🖼️ **批量处理** — 拖放或选择多张图片，一键压缩
- 🎚️ **质量调节** — 1-100% 滑块，实时调节压缩强度
- 🔄 **格式转换** — 保持原格式 / 转为 JPEG / PNG / WebP
- 📐 **智能缩放** — 可选最大宽高限制，不放大原图
- 👁️ **并排预览** — 原图与压缩后效果对比
- 📊 **压缩统计** — 前后体积对比、节省百分比
- 🔒 **变大保护** — 如果压缩后更大且无缩放，自动保留原图
- 💾 **灵活输出** — 覆盖原文件 / 同目录加后缀 / 自定义目录

## 触发方式

| 触发 | 说明 |
|------|------|
| `图片压缩` / `compress` / `批量压缩` / `压缩图片` | 打开主界面 |
| 在启动器中选中图片 → 搜索上述关键词 | 带文件进入 |
| 图片右键 / 文件触发 | 带文件进入 |

## 使用说明

1. 输入关键词打开插件
2. 拖放图片到窗口，或点击选择图片
3. 调整压缩参数（质量、格式、尺寸、输出位置）
4. 点击 **压缩** 预览效果
5. 满意后点击 **应用** 保存

## 支持的格式

- 输入：JPEG, PNG, WebP, GIF, AVIF, TIFF, BMP, SVG, ICO
- 输出：JPEG, PNG, WebP（或保持原格式）
