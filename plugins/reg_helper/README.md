# 正则表达式编辑器 (reg_helper)

> JavaScript 正则表达式编辑器，支持实时高亮匹配、替换预览、常用预设和速查表。

## 功能特性

- **实时匹配高亮** - 输入正则后即时在测试文本中高亮所有匹配，使用交替颜色区分不同匹配项
- **匹配详情** - 显示每个匹配的索引、长度和捕获组（包括命名捕获组）
- **替换预览** - 支持输入替换文本，实时预览替换结果
- **标志切换** - 工具栏直接切换 `g` `i` `m` `s` `u` 标志
- **预设模式** - 内置邮箱、URL、手机号、身份证、IP 等 10 种常用正则，一键应用
- **正则速查表** - 按「字符类/量词/锚点/分组」分类的速查面板，点击即可插入
- **深色/浅色主题** - 自动跟随 Mulby 主题切换
- **选中文本直接匹配** - 支持 `over` 触发器，选中文本直接进入编辑器

## 触发方式

| 触发 | 说明 |
|------|------|
| `regex` | 关键字打开编辑器 |
| `正则` | 关键字打开编辑器 |
| 选中任意文本 | 将选中文本自动填入测试文本区 |

## 快捷操作

- **复制正则** - 将当前正则（含标志）复制到剪贴板
- **复制匹配** - 将所有匹配结果按行复制
- **复制替换结果** - 复制替换后的完整文本
- **重置** - 清空所有输入
