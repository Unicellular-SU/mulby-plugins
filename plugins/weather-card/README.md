# weather-card

一键天气卡片 Mulby 插件：输入城市名或使用当前位置，生成当前天气与未来三日预报卡片，并支持复制为 Markdown 或纯文本。

## 功能特性

- 🔎 城市搜索：支持中英文城市名，全球任意地区可查。
- 📍 当前位置：调用 Mulby 定位 API 获取经纬度，优先显示实际位置名称，无法解析名称时显示经纬度；成功定位后会保存位置，下次打开自动查询。
- 🌡️ 当前天气：展示温度、体感温度、湿度、风速、能见度。
- 📅 三日预报：展示未来三天最高/最低温度与天气图标。
- 🌓 明暗适配：跟随 Mulby 主题切换浅蓝明亮模式 / 深邃暗黑模式。
- 📋 卡片复制：支持复制 Markdown 与纯文本两种格式。

## 触发方式

在 Mulby 主输入框输入关键词：

- `天气` - 打开天气卡片工具

## 使用示例

1. 输入 `天气` 打开插件。
2. 输入 `北京`、`London`、`Tokyo` 等城市名并点击“生成卡片”。
3. 或点击“当前位置”，按提示授予定位权限后自动生成所在地天气；插件会保存本次定位，下次打开自动查询。
4. 查询成功后可点击“复制 Markdown”或“复制纯文本”导出卡片内容。

## 权限与依赖

- 需要网络访问 Open-Meteo 天气与地理编码服务。
- 使用 Mulby `clipboard` 权限复制卡片内容。
- 使用 Mulby `notification` 权限展示失败提示。
- 使用 Mulby `geolocation` 权限查询当前位置天气。
