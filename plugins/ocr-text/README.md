# OCR 文字识别

极便捷的 OCR 识别工具，支持提取文字、表格、公式。

## 功能特性

- **截图文字识别** - 一键截图提取文字内容
- **截图文字识别+复制** - 识别文字并自动复制到剪贴板
- **截图文字识别+翻译** - 识别文字并自动翻译
- **截图表格识别** - 识别图片中的表格，输出 Markdown 表格格式
- **截图公式识别** - 识别数学公式，输出 LaTeX 格式
- **图片文字识别** - 拖入图片文件进行识别
- **插件内截图/选图** - 打开插件后可直接点击“截图识别”或“选择图片”重新识别
- **表格结果导出** - 支持复制 Markdown、复制可粘贴到 Excel 的 TSV，以及导出 `.xls`
- **公式结果导出** - 支持复制 LaTeX，以及导出 SVG 公式图片

## 触发方式

| 功能 | 关键词 |
|------|--------|
| 文字识别 | `OCR` / `文字识别` / `ocr` |
| 文字识别+复制 | `OCR复制` / `文字识别复制` |
| 表格识别 | `表格识别` / `OCR表格` |
| 公式识别 | `公式识别` / `OCR公式` |
| 文字识别+翻译 | `OCR翻译` / `文字识别翻译` |
| 图片识别 | 拖入 PNG/JPG/JPEG/BMP/GIF/WebP/TIFF 图片 |

## 使用说明

1. 在 Mulby 搜索框输入对应关键词
2. 截取需要识别的屏幕区域
3. 也可以在插件窗口内点击“截图识别”或“选择图片”重新选择输入
4. 等待识别完成
5. 点击复制按钮复制识别结果
6. 切换到表格模式后，可复制 Markdown、复制 Excel 格式或导出 `.xls`
7. 切换到公式模式后，可复制 LaTeX 或导出 SVG 公式图片
8. 可切换文字/表格/公式模式重新识别

## 快捷键

可在 Mulby 设置中为各功能绑定全局快捷键，实现一键调用。

## 依赖

- 需要配置 AI 模型（支持视觉识别的模型，如 GPT-4o、Claude 等）
