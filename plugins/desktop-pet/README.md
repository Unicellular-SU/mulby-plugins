# 桌面宠物 🐱

一只活泼的桌面伴侣，能感知你的鼠标和键盘操作并做出有趣反应。

## 功能特性

- **追逐光标** — 鼠标远离时，小猫会跑向光标位置
- **好奇张望** — 鼠标在附近缓慢移动时，小猫会转头观察
- **被点击跳起** — 左键点击小猫附近，它会开心地跳起
- **被右键惊吓** — 右键点击附近，小猫会受惊
- **闲逛** — 无操作 10 秒后随机走动
- **打瞌睡** — 无操作超过 60 秒会坐下，继续无操作则打瞌睡
- **鼓掌加油** — 连续快速打字时，小猫为你加油
- **摇晃** — 滚动鼠标滚轮时，小猫被"风"吹得摇晃
- **庆祝** — 按下 ⌘+S (保存) 时撒花庆祝

## 使用方式

在 Mulby 搜索栏输入以下任意关键词启动：

- `宠物`
- `pet`
- `桌面宠物`

## 自定义宠物外观

右键点击宠物 → 选择「设置外观...」，打开配置面板：

1. 点击上传区域选择任意图片（PNG/JPG/WebP/GIF）
2. 插件自动将图片转为像素画风格
3. 调整像素精度（8×8 ~ 32×32）
4. 点击「保存并应用」即时生效

也可以点击「恢复默认」回到内置橘猫角色。

## 右键菜单

右键点击宠物弹出菜单：
- 🎨 设置外观... — 打开自定义面板
- 👋 加油！— 触发鼓掌动画
- 😴 休息一下 — 进入睡眠状态
- 👁 暂时隐藏 — 隐藏宠物窗口
- ❌ 关闭宠物 — 关闭插件

## 技术实现

- 120x120 透明置顶窗口
- Canvas 程序化绘制像素风格小猫（无需外部图片资源）
- 自定义图片→像素画转换引擎（智能调色板提取 + 最近颜色匹配）
- 全局输入监听 (inputMonitor) 感知鼠标/键盘
- 行为决策状态机驱动 9 种行为状态
- 15 种动画状态（idle、walk、run、jump、sit、sleep 等）
- 子窗口设置面板（暗色主题 UI）
- Storage 持久化用户自定义精灵数据

## 权限

- `inputMonitor` — 全局鼠标和键盘输入监听（macOS 需要辅助功能权限）

## 系统要求

- Mulby 桌面客户端
- macOS 需授予辅助功能 (Accessibility) 权限
