# 单位转换插件

一个 Mulby 核心版数值转换插件，提供 10 类单位换算、在线汇率、历史记录与导出能力，界面采用深蓝玻璃拟态风格。

## 功能说明

- 支持转换类别：
  - 长度、重量、温度、货币、数据、时间、面积、体积、速度、进制
- 实时转换：
  - 输入即算，支持科学计数法（如 `1.2e6`）
  - 智能识别输入中的单位后缀（如 `100km`）
- 货币转换：
  - 在线汇率更新（每小时自动更新）
  - 自动加载汇率接口支持的全部币种（由 API 返回决定）
  - 手动刷新按钮
  - 离线或刷新失败时自动回退缓存并提示
- 历史记录：
  - 最近 10 条
  - 一键重做
  - 清空历史
  - 导出 JSON / CSV（复制到剪贴板）
- 批量模式：
  - 每行输入一个值并批量生成结果

## 触发方式

- 关键词：
  - `单位换算`
  - `单位转换`
  - `数值转换`
  - `converter`
- 划词菜单：
  - `转换数值`

## 使用示例

- `100km` -> 选择目标单位 `mile`
- `32f` -> 目标单位 `C`
- 进制模式中输入 `FF`（from = HEX）-> 目标单位 `DEC`

## 构建与打包

```bash
pnpm install
pnpm run build
pnpm run pack
```

## 依赖与前置条件

- 依赖 Mulby 插件运行环境（`mulby dev` / `mulby pack`）
- 货币汇率依赖在线接口：`https://api.frankfurter.dev/v1`
