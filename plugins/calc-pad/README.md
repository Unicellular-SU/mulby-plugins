# 计算稿纸 (Calc Pad)

高效简洁的计算工具，实时计算、自动保存，像稿纸一样记录每一步过程。

## 特性

- **实时计算**：输入算式即刻获得结果。
- **自动保存**：所有稿纸自动持久化，无需手动保存。
- **行间备注**：可以为任意计算行添加说明备注。
- **多稿纸管理**：随时新建、切换、删除多个稿纸。
- **丰富运算支持**：支持 `+`、`-`、`*`、`/`、`^`、`%` 和括号嵌套，提供千分位格式化显示。
- **键盘驱动**：
  - `Enter`：完成当前输入，如无输入则将上一行结果带入新行
  - `↑` / `↓`：在各计算行间切换焦点，支持随时修改历史算式
  - `Cmd/Ctrl + B`：为当前行添加/编辑备注
  - `Cmd/Ctrl + C`：快速复制当前所有算式和结果
  - `Cmd/Ctrl + N`：新建空白稿纸
  - `Cmd/Ctrl + R`：清空当前稿纸

## 唤起方式

在 Mulby 中输入关键字：`计算稿纸`、`calc`、`计算器`

## 示例

```
55+88+7689*543
                       = 4,175,270
                       
2^5+888+99*878.467
                       = 87,888.233
```
