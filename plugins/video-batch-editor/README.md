# 视频批量编辑

Mulby 插件：面向本地视频文件的批量处理工作台。使用 Mulby 宿主管理的内置 FFmpeg 运行时执行本地视频任务。

## 当前能力

- 批量导入视频文件，支持从 Mulby 文件附件启动。
- 扫描导入文件夹，递归收集常见视频文件并统计跳过项。
- 检测、下载并使用 Mulby 内置 FFmpeg。
- 校验常见视频扩展名。
- 配置导出预设：MP4/H.264、MP4/H.265、WebM/VP9、封面 JPG。
- 配置批量截取：完整视频、指定片段、截取开头、去掉开头。
- 配置画面裁剪：手动裁剪、中心 1:1、中心 9:16、中心 16:9。
- 配置横竖屏互转：横屏 16:9、竖屏 9:16、方屏 1:1、左右旋转。
- 配置分辨率、码率、CRF 和文字水印。
- 生成 FFmpeg 命令预览和输出路径。
- 串行执行批量队列，显示 FFmpeg 进度，支持停止当前任务和失败重试。
- 处理完成后可定位输出文件，并导出 JSON 处理日志。

当前版本先做本地批处理的核心闭环：导入文件/文件夹、批量截取、画面裁剪、横竖屏转换、生成命令、执行队列、错误恢复、输出定位和日志导出。后续可继续补模板保存和更丰富的视频处理模板。

## 触发方式

| 方式 | 说明 |
| --- | --- |
| `视频批量` | 打开视频批量编辑工作台 |
| `批量视频` | 打开视频批量编辑工作台 |
| `视频批量编辑` | 打开视频批量编辑工作台 |
| `video batch` | 打开视频批量编辑工作台 |
| 文件附件 | 从选中的文件进入工作台并预填队列 |

## 前置条件

需要 Mulby 宿主提供 `window.mulby.ffmpeg` API。首次使用时可在插件内点击“下载 FFmpeg”，由 Mulby 按当前系统下载并解压运行时。

## 开发

```bash
cd plugins/video-batch-editor
pnpm install
pnpm run build
pnpm run pack
```
