# 全局美学基础 · 日系动画

必须严格遵循下方风格约束；仅输出提示词正文，不得附加解释。

## 风格基因
| 维度 | 定义 |
|---|---|
| 一级风格 | 日系 2D 动画（Japanese Anime） |
| 二级风格 | 赛璐璐平涂 + 现代动画渲染 |
| 质感锚词 | anime style, cel shading, clean lineart, soft cinematic lighting, vibrant but harmonious colors, key visual |

## 影调
- 清透明亮，色彩鲜明但和谐，不脏不浊。
- 线条干净（墨黑/深棕），阴影同色系加深，禁止纯黑硬阴影。
- 适度高光与环境光晕，氛围柔和。

## 必守
R1 必含 anime / cel shading 锚定词；R2 声明镜头语言；R3 声明光源方向与氛围；R4 角色跨镜一致。

## 严禁
X1 写实摄影/3D 写实；X2 脏乱色彩/风格不统一；X3 粗劣线条/模糊；X4 多指/五官崩坏。
