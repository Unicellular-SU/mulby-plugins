# 全局美学基础 · 写实电影感

必须严格、完整遵循下方全部风格约束与全局规则，并严格按提示词模板格式生成提示词；仅输出提示词正文，不得附加任何解释、说明、注释、标题或其他额外文本。

## 一、风格基因

| 维度 | 定义 |
|---|---|
| 一级风格 | 写实电影感（Cinematic Photoreal） |
| 二级风格 | 真人实拍 · 电影摄影 · 自然主义 |
| 情感基调 | 真实可信、克制内敛、有呼吸感的光影 |
| 质感锚词 | photorealistic, cinematic still, 35mm film, shallow depth of field, natural lighting, filmic color grade |

## 二、全局色彩与影调（风格基线）

| 层级 | 约束强度 | 说明 |
|---|---|---|
| L1 硬约束 | 高 | 自然肤色、可信材质、电影级动态范围 |
| L2 软约束 | 中 | 调色板按场景/情绪在自然范围内微调 |
| L3 例外 | 低 | 强戏剧时刻可加强对比/色偏 |

### 调色倾向
- 整体偏电影感：略微压暗高光、保留暗部细节、轻微胶片颗粒。
- 肤色自然不偏黄/不过曝；中性到微暖白平衡（5200–5600K）。
- 对比中等偏高，饱和中等（避免数码味的高饱和）。

### 情绪光影对照

| 情绪基调 | 光线类型 | 调色/对比 |
|---|---|---|
| 温情/日常 | 柔和自然光、窗光、黄金时刻暖光 | 暖调、低对比、柔和阴影 |
| 紧张/悬疑 | 硬光侧逆光、低照度、明暗强切 | 冷调、高对比、大面积阴影 |
| 悲伤/孤独 | 漫射阴天光、冷蓝时刻 | 去饱和、低对比、冷调 |
| 高光/胜利 | 逆光轮廓光、丁达尔光束 | 暖金、高光晕染、通透 |
| 回忆/梦境 | 柔焦溢光、雾化扩散 | 提亮暗部、淡雾、低对比 |
