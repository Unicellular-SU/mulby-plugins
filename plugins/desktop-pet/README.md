# 桌面宠物

一只住在桌面上的像素小幽灵。它能根据鼠标、键盘、前台应用、番茄钟、提醒和对话上下文做出表情、姿势、气泡与 AI 回复。

## 功能特性

- **桌面陪伴**：透明置顶小幽灵，会闲逛、追逐、探头、睡觉、挥手、跳跃和摇摆。
- **AI 对话**：长按宠物打开输入框；宠物会结合性格、心情、生活档案、天气和当前应用上下文回复。
- **表现控制**：AI 可调用表情、姿势、动作、移动和心情工具，让回复过程同步变脸和运动。
- **生活档案**：多轮对话后自动维护用户档案、偏好、习惯、关系线索和近期事件，可在设置页编辑或清空。
- **生态系统**：记录精力、关注、好奇、专注、补水状态，识别工作模式，维护今日时间线和每日目标。
- **日记时间线**：晚上自动根据今日事件、最近对话和统计数据生成宠物日记。
- **番茄钟与节律提醒**：支持专注计时、喝水提醒、休息提醒和自定义定时提醒。
- **小游戏**：支持猜谜语、成语接龙、冷知识问答，并记录积分、连续答对和今日游玩情况。
- **成就与状态**：记录亲密度、心情、签到、互动、专注时长和成就解锁。

## 使用方式

在 Mulby 搜索栏输入以下任意关键词启动：

- `宠物`
- `pet`
- `桌面宠物`

## 操作

- 左键点击：触发宠物互动与主动搭话。
- 长按左键：打开聊天输入框。
- 右键点击：打开菜单，可进入设置、开始番茄钟、启动小游戏、查看思考过程或退出。

## 设置页

右键宠物后选择「设置」：

- **设置**：宠物名称、性格、AI 模型、说话频率、番茄钟时长、生日、位置信息、上下文隐私和触发行为。
- **生态**：查看心情、亲密度、签到、番茄钟、专注时长、需求状态、当前工作模式、每日任务、小游戏积分和今日时间线。
- **提醒**：配置喝水、休息眼睛和自定义定时提醒。
- **记忆**：查看、编辑、删除或清空生活档案。
- **成就**：查看已解锁成就。
- **日记**：查看自动生成的宠物日记。
- **对话**：查看用户上下文、模型思考和宠物发言历史。
- **表现预览**：测试所有表情、姿势、情绪和动作组合。

## 隐私说明

- 默认只使用应用名识别工作模式，不把窗口标题写入长期存储。
- 设置页可选择允许窗口标题参与短期 AI 上下文；标题仍不会进入今日时间线。
- 剪贴板响应默认关闭，并沿用敏感内容过滤。

## 技术实现

- 透明置顶 detached 窗口，内容区为 `PET_SIZE` 120。
- 内置 SVG 小幽灵精灵，由 `svg-renderer` 加载，姿势和表情通过状态机与 CSS 动画表现。
- 输入监听使用 Mulby `inputMonitor`，前台窗口上下文通过主进程 RPC 读取。
- 数据使用 Mulby storage 持久化，包括性格、位置、生活档案、对话历史、生态状态、时间线、每日任务、小游戏统计、日记和成就。

## 权限

- `inputMonitor`：监听鼠标和键盘行为。
- `ai`：调用文本模型生成回复、日记和小游戏题目。
- `accessibility`：在 macOS 上支持全局输入监听。
- `clipboard`：可选剪贴板响应。
- `screen`：读取屏幕与窗口位置。
- `geolocation`：可选定位和天气上下文。

## 系统要求

- Mulby 桌面客户端。
- macOS 需授予辅助功能权限。
